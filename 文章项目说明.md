我在公众号和知乎上持续输出两个系列的内容：**理财访谈**和**KZCQL**。前者面向理财新手分享实战经验，后者记录我用 AI 多智能体协作系统进行写作的完整探索过程。两个系列的共同点是——都追求"真实经历 + 可复用方法论"，不贩卖焦虑，只输出自己真正踩过坑、验证过的东西。

## 四、写作方法论

我的写作流程建立在 KZCQL 多智能体协作系统之上，每篇文章都经过完整的流水线处理：

```
W1 撰写 → R1 事实核查 → R2 全量评审 → R4 综合评级 → W2 迭代修改
```

这套流程的核心是一套 **12 维度 100 分制评审体系**，覆盖事实准确性、逻辑严密性、可读性、原创性等维度，确保每篇文章在发布前达到既定质量标准。

**风格定位**：有见识的普通人在认真聊一件打动他的事。
