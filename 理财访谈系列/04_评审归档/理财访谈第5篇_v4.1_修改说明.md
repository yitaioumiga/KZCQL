### 修改1：场景1（宏观动荡）的分级机制

**插入位置**: 第77行（第一个触发场景）

**修改前**:
```
第一个，宏观动荡。中东、俄乌、台海、朝鲜半岛、南海这些敏感区域有没有重大地缘冲突。如果有，启动防御模式，建议降低风险敞口。这个听起来离我们很远，但你想想今年国际局势那个折腾劲儿，真出了事，你的基金账户不会因为你不看就没事。
```

**修改后**:
```
第一个，宏观动荡。中东、俄乌、台海、朝鲜半岛、南海这些敏感区域有没有重大地缘冲突。这个我设计了一个升级机制。

**🟡 关注级别**：有事件但无直接军事冲突。比如谈判破裂、制裁升级、能源价格波动。这时候系统会标记关注，建议核心资产保持定投坚决不卖，卫星资产持有不动。

**🟠 紧张级别**：关注状态持续超过30天，或者出现新的升级信号（比如某国宣布进入战争状态）。这时候系统会自动提升关注级别，建议你开始评估是否需要降低风险敞口。

**🔴 警戒级别**：出现大国直接军事冲突。这时候系统会启动防御模式，建议你考虑降低风险敞口，比如暂停新增定投，或者把部分股票资产转移到债券或黄金。

为什么要加30天自动升级？因为我跑了数据之后发现，中东局势从4月到现在一直处于「有事件但无直接冲突」的状态，如果一直处于同一级别，你可能慢慢就习惯了，真正的升级来的时候反而反应迟钝。自动升级机制就是防止这种「关注疲劳」。
```

### 修改2：场景5（估值泡沫预警）的分级机制

**插入位置**: 第93行（第五个触发场景，因前序插入行号已变化）

**修改前**:
```
第五个，估值泡沫预警。PE百分位有没有超过80%。如果超过了，仅提醒关注，不强制操作。这个比较特殊，后面会详细说为什么。
```

**修改后**:
```
第五个，估值泡沫预警。这个比较特殊，我设计了一个分级机制。

**🟡 关注级别**：PE百分位超过80%，但持仓收益低于15%。这时候系统只会提醒你关注，不强制操作。因为PE高不代表明天就跌，可能还会继续涨。

**🟠 预警级别**：PE百分位超过80%且持仓收益达到15%以上，或者PE百分位直接超过90%。这时候系统会建议你放缓定投节奏，比如把每月的定投金额减半。

**🔴 警戒级别**：PE百分位超过95%且持续10天以上。这时候系统会建议你考虑暂停定投，甚至部分止盈。

为什么要加持仓收益这个条件？因为我跑了30天数据之后发现，PE百分位从4月到现在几乎没低于过80%，如果只看这一个指标，系统会一直处于触发状态，预警就丧失了区分度。但加上持仓收益之后，只有当市场真的涨了很多、你账面上有浮盈的时候，才会触发更高级别的响应。这样既不会错过真正的泡沫，也不会被常驻触发搞得麻木。
```
