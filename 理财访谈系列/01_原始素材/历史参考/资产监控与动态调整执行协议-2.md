# 资产监控与动态调整执行协议

## (Asset Monitoring & Adjustment Protocol)

**版本：** V1.0 **生效日期：** 2025年10月 **适用对象：** 川大ACCA学生定投计划

### 一、 协议目的

本协议旨在建立一套机械化、去情绪化的投资纪律，明确规定**何时检查（周期）以及遇到特定情况如何行动（触发机制）**，以确保“核心-卫星”策略的长期有效执行。

### 二、 常规监控周期 (Routine Check)

#### 1. 月度检查 (Monthly)

- **时间点：** 每月定投扣款日的前后3天（推荐：每月15日-18日）。
    
- **检查内容：**
    
    - **账户资金：** 确认绑定银行卡余额 $\ge 500$ 元（确保扣款成功）。
        
    - **卫星状态：** 扫一眼“卫星”基金（目前为AI和军工）的持有收益率。
        
    - **黄金价格：** （仅在过渡期）查看金价是否回调至心理价位（如 $\le 880$ 元/克）。
        
- **标准动作：**
    
    - 若一切正常 $\rightarrow$ **不做任何操作**，让系统自动扣款。
        
    - 若黄金回调到位 $\rightarrow$ **启动调整**（将AI/军工的部分额度切回黄金）。
        

#### 2. 季度/半年度检查 (Quarterly/Semi-Annually)

- **时间点：** 每季度末（3月、6月、9月、12月）的最后一周。
    
- **检查内容：**
    
    - **估值水位：** 登录理财App查看沪深300和中证500的**PE（市盈率）百分位**。
        
    - **宏观环境：** 阅读一份季度的宏观策略报告（或向调研员索要），关注“GDP增速”和“货币政策松紧”。
        
- **标准动作：**
    
    - 若 PE百分位 $< 20\%$（低估） $\rightarrow$ 考虑利用节余资金**追加定投**（如 $+100$ 元/月）。
        
    - 若 PE百分位 $> 80\%$（高估） $\rightarrow$ 考虑**减少定投**（如 $-100$ 元/月）。
        
    - 若处于 20%-80% 之间 $\rightarrow$ **维持现状**。
        

#### 3. 年度检查 (Yearly)

- **时间点：** 每年寒假（1月或2月）。
    
- **检查内容：** **资产再平衡 (Rebalancing)**。计算各大类资产的实际占比。
    
- **标准动作：**
    
    - 若权益类资产（股票基金）占比 $> 40\%$（当前目标约为30%-40%） $\rightarrow$ **卖出**超额部分的股票基金，**买入**债券基金。
        
    - 若权益类资产占比 $< 20\%$ $\rightarrow$ 将部分到期的债券基金赎回，**买入**股票基金（或加大定投）。
        

### 三、 事件驱动触发与应对 (Event-Driven Triggers)

当发生以下特定事件时，打破常规周期，**立即启动**应对程序。

| 触发场景 (Trigger)   | 识别指标 (Thresholds)                                           | 应对策略 (Action)                                                                                                                                                   |
| ---------------- | ----------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. 宏观动荡/战争爆发** | 官方新闻确认大国军事冲突、地缘局势严重恶化                                       | **启动防御模式：**<br><br>1. **核心资产：** 保持定投，**坚决不卖**。<br><br>2. **卫星资产：** 立即评估**黄金**和**军工**。若未配置，从生活费结余中切分 $50-100$ 元/月启动定投；若已配置，持有不动。                                 |
| **2. 股市崩盘/大跌**   | 沪深300指数<br><br>**单周跌幅 > 10%**<br><br>或 **累计跌幅 > 20%**       | **启动“微笑曲线”收集模式：**<br><br>1. **心理建设：** 默念“这是打折促销”。<br><br>2. **行动：** 维持定投金额不变。若手头有额外闲钱（如奖学金），可一次性**追加买入 500-1000 元**。**严禁停止扣款。**                                 |
| **3. 卫星资产收益达标**  | AI / 军工 / 黄金<br><br>**持有收益率** $\ge 30\%$                    | **启动止盈模式：**<br><br>1. **分批止盈：** 卖出该基金持仓市值的 **50%**。<br><br>2. **落袋为安：** 卖出资金转入**稳健债券基金**（如兴业稳利/易方达兴利），作为储备金锁定利润。<br><br>3. **剩余持仓：** 继续持有，若收益率回撤至 $20\%$，则全部卖出。 |
| **4. 资产比例极度失衡**  | 股票类资产市值占比<br><br>$\ge 80\%$<br><br>_(注：当前占比仅约10%，此触发需长期积累)_ | **启动强制再平衡：**<br><br>说明市场极度过热。**卖出**超配的股票基金，**买入**债券基金，将股票占比强制回调至 **40%-50%**。                                                                                   |
| **5. 估值泡沫**      | 沪深300 PE百分位<br><br>$\ge 80\%$<br><br>_(进入历史高估区)_            | **启动防御性减仓：**<br><br>1. **减少定投：** 将每月定投金额**减半**（$500 \rightarrow 250$ 元）。<br><br>2. **分批卖出：** 每月卖出核心资产持仓的 **5%-10%**，转入货币基金“过冬”。                                 |
| **6. 债市大跌**      | 债券基金净值<br><br>**单日跌幅 > 0.3%**<br><br>或 **周跌幅 > 1%**         | **启动久期调整：**<br><br>说明利率环境恶化（如加息）。检查手头债基，若是长债基金（如30年国债），考虑转换成**短债基金**（如30天/90天持有期）或**货币基金**以规避波动。                                                                |
|                  |                                                             |                                                                                                                                                                 |

### 四、 执行承诺

本人已充分理解上述协议内容，承诺在未来的投资过程中：

1. **尊重规则：** 不以主观情绪（恐慌/贪婪）替代客观规则。
    
2. **严格执行：** 无论市场涨跌，只要未触发调整条件，就坚持原定计划。
    
3. **定期复盘：** 每年至少进行一次全面复盘，根据自身财务状况的变化（如毕业工作、收入增加）修订本协议的参数。
