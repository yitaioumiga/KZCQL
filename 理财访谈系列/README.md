## 访谈核心要点

### 人物背景
- **起点**：从价值投资转向定投（财报分析太耗时、A股不稳定）
- **当前状态**：定投约1.5年，月均收益率约1%，本年至今约2%
- **痛点**：之前炒股/黄金投资被波动折磨，影响心情和工作

### 策略架构

```
┌─────────────────────────────────────────┐
│           资产配置（当前）                 │
├─────────────────────────────────────────┤
│  长期债券（易方达纯债）    约 85%        │
│  定投基金组合              约 12.8%      │
│    ├── 核心：沪深300 + 中证500          │
│    └── 卫星：AI + 军工                  │
└─────────────────────────────────────────┘
```

### 智能体应用
- **平台**：扣子（Coze）
- **形态**：Skill（技能）
- **检查周期**：每日 + 每月 + 每季度
- **输出**：QQ邮箱报告，每天1-2分钟查看

### 每日检查6维度
1. 股票估值泡沫预警
2. 股票崩盘预警
3. 债市崩盘预警
4. 宏观动荡预警
5. 收益达标检查（≥20%止盈）
6. 资产比例检查（股票≤80%）

## 写作注意事项

1. **收益率表述**：强调"约"、"目前"，避免承诺
2. **基金名称**：易方达纯债等具体名称可提但不做推荐
3. **风险提示**：定投不保本，过往收益不代表未来
