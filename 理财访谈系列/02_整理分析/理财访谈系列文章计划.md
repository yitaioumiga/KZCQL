# 理财访谈系列文章计划

> **系列定位**：面向18-25岁理财新手的个人理财经验分享
> **核心价值**：真实经历 + 可复用方法论 + 实践方案
> **发布节奏**：每周1篇，共12篇（约3个月）
> **文章篇幅**：3000-5000字

---

## 一、系列整体框架

### 1.1 核心叙事线

```
踩坑经历 → 转型思考 → 策略搭建 → 自动化实现 → 持续优化
```

**三阶段递进**：
- **第一阶段（第1-4篇）**：为什么普通人需要定投？（认知升级）
- **第二阶段（第5-8篇）**：如何搭建适合自己的定投体系？（方法论）
- **第三阶段（第9-12篇）**：如何让定投更省心？（自动化与优化）

### 1.2 受众画像

| 特征 | 描述 |
|------|------|
| 年龄 | 18-25岁大学生/职场新人 |
| 收入 | 每月可支配500-2000元 |
| 痛点 | 没时间盯盘、被炒股/基金折磨过、不知道从哪开始 |
| 需求 | 省心、低门槛、可复用的方案 |

### 1.3 系列核心方法论

```
┌─────────────────────────────────────────────────────┐
│              普通人省心理财体系                        │
├─────────────────────────────────────────────────────┤
│  核心策略：定投 + 核心卫星配置 + 股债平衡              │
│  执行工具：智能体自动化检查（每日/月/季度）             │
│  时间投入：每天1-2分钟看报告                          │
│  预期收益：月均约1%（历史数据，不作承诺）               │
└─────────────────────────────────────────────────────┘
```

#### 第1篇：开篇——从价值投资到定投，我花了1.5年找到的普通人最优解

**发布时间**：第1周

**核心主题**：踩坑经历 + 转型思考

**文章结构**：
1. **引子**：一个普通大学生的理财困惑
2. **踩坑史**：
   - 价值投资的门槛：财报分析耗时、A股不稳定、资本操盘难以预测
   - 炒股的折磨：波动大、影响心情、占用学习工作时间
   - 黄金投资的教训：小额尝试也被波动折磨
3. **转型契机**：为什么选择定投？
   - 不需要择时
   - 长期坚持能平摊成本
   - 适合没时间盯盘的人
4. **核心观点**：定投不是赚最多的方法，但可能是最适合普通人的方法
5. **预告**：下篇详细讲我的定投策略

**方法论要点**：
- 价值投资 vs 定投的对比分析
- 普通人理财的3个核心约束：时间、精力、专业知识

**可复用实践**：
- 自我评估清单：你适合价值投资还是定投？
- 时间投入对比表

**数据支撑**：
- 定投时长：约1.5年
- 月均收益率：约1%

#### 第2篇：定投的7个致命缺陷，我全踩过

**发布时间**：第2周

**核心主题**：定投不是万能药，正视缺陷才能用好它

**文章结构**：
1. **引子**：定投被吹得太神了
2. **7个缺陷逐一拆解**：
   - 缺陷1：单边牛市中的"收益拖累"
   - 缺陷2："钝化效应"——后期威力大减
   - 缺陷3：令人绝望的"苦笑曲线"
   - 缺陷4：资金利用率低
   - 缺陷5："止盈"比"坚持"更难
   - 缺陷6：无法规避"烂资产"风险

3. **我的应对策略**：
   - 承认并接受"防守成本"
   - 资产再平衡机制
   - 止盈纪律
   - 精选宽基指数
4. **核心观点**：定投有缺陷，但这些缺陷并非不可战胜

**方法论要点**：
- 每个缺陷的应对策略

5. **核心观点**：体系化思维，让规则替代情绪

**方法论要点**：

## 四、写作注意事项

### 4.1 风险提示规范

1. **收益率表述**：强调"约"、"目前"，避免承诺
2. **基金名称**：可提具体名称但不做推荐
3. **风险声明**：定投不保本，过往收益不代表未来
4. **平台提及**：扣子（Coze）是字节产品，客观描述

### 4.2 内容风格要求

1. **真实感**：使用访谈中的真实对话和经历
2. **可操作**：每篇文章提供可复用的实践方案
3. **数据支撑**：关键结论用数据说话
