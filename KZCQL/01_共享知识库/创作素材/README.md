## 多层次索引体系

```
Level 0（粗索引）：素材索引.md
  → 按主题/时间/类型分类，定位大方向
  ↓
Level 1（细索引）：随笔素材/细索引/F[n]_摘要索引.md
  → 按时间段/主题段落摘要，定位具体段落和行号
  ↓
Level 2（原文）：随笔素材/F[n].md
  → 原始日志内容，按需查阅具体段落
```

## 使用方式

1. **选题阶段**：先查 `选题分析与创作计划/内容矩阵与主题边界.md`，确认不重复
2. **素材定位**：先查 `素材索引.md`（粗索引）→ 再查 `细索引/`（细索引）→ 最后查阅原文
3. **外部资产**：查 `外部资产索引/` 获取开源项目和个人网站信息
4. **定位相关**：查 `一人公司定位资料包/01_核心报告/`
5. **风格参考**：查 `作者偏好与风格档案.md`

## 注意事项

- 随笔素材文件较大（共约2.5MB），**不要全文加载**，按索引逐段查阅
- 细索引已标注⭐高价值素材（共71处），优先使用
- 素材索引会持续更新，新增素材后请同步更新索引
- 随笔中的内容为原始记录，可能存在口语化、不完整的情况，使用时需改写提炼
