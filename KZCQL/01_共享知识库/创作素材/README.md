# 创作素材库使用说明

> 本目录存放以太公众号内容创作的核心素材，供W1/D2/D3等智能体查阅。

## 目录结构

```
创作素材/
├── README.md              ← 你正在阅读的文件
├── 素材索引.md            ← 按主题/时间/类型分类索引（智能体优先查阅此文件）
├── 随笔素材/              ← 随笔整合四件套（F1-F4，共约250万字）
│   ├── F1-幸福五日志_整合版.md
│   ├── F2-随笔整合_2025.09.10-11.27.md
│   ├── F3-随笔整合_2025.11.28-2025.4.6.md
│   └── F4-随笔总整合_2025.01.25-2026.05.21.md
└── 一人公司定位资料包/      ← 更新版定位报告（第八轮+补充）
    ├── 01_核心报告/
    ├── 02_阶段归档/
    ├── 03_补充分析/
    ├── 04_参考资料/
    ├── 05_指导手册/
    └── 06_原始资料/
```

## 使用方式

1. **智能体查阅素材时**，先读取 `素材索引.md`，按主题/时间/类型定位相关素材
2. **定位相关内容**，查阅 `一人公司定位资料包/01_核心报告/`
3. **个人经历和故事**，查阅 `随笔素材/F1-F4`
4. **能力画像和透镜**，查阅 `作者事实档案.md` §5

## 注意事项

- 随笔素材文件较大（共约2.5MB），请按需查阅具体段落，不要全文加载
- 素材索引会持续更新，新增素材后请同步更新索引
- 随笔中的内容为原始记录，可能存在口语化、不完整的情况，使用时需改写提炼
