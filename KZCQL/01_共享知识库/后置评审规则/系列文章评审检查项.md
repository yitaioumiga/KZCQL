# 系列文章评审检查项

> 系列文章专属评审规则。R2 在评审系列文章时，除常规 10 维度评分外，必须额外检查本文件中的项目。
>
> 适用范围：系列文章评审（非系列文章不适用）
> 最后更新：2026-05-18
>
> **P23补丁说明**：新增 S-08 风险提示完整性，适用于理财访谈系列和投资理财类文章

---

## 使用说明

本检查项嵌入 R2 全量评审流程，作为 D4（结构完整性）的补充检查。

- 系列文章：执行本文件全部检查项
- 非系列文章：跳过本文件

---

## 检查项（6项）

### S-01 独立可读性

| 属性 | 内容 |
|------|------|
| **编号** | S-01 |
| **名称** | 独立可读性 |
| **所属维度** | D4 结构完整性 |
| **判定标准** | 文章是否存在仅对系列老读者有意义、新读者无法理解的内容 |
| **通过条件** | 无「上期讲过就不重复了」「关注本系列的朋友知道」等假设读者看过前文的表述 |
| **扣分** | 每处扣 1 分（D4 维度） |

### S-02 下期预告

| 属性 | 内容 |
|------|------|
| **编号** | S-02 |
| **名称** | 下期预告 |
| **所属维度** | D4 结构完整性 |
| **判定标准** | 文章末尾是否有下期预告 |
| **通过条件** | 金句清单之后有「下期预告」段落，包含下一期主题和悬念/钩子 |
| **扣分** | 缺失扣 2 分（D4 维度） |

### S-03 往期概念说明

| 属性 | 内容 |
|------|------|
| **编号** | S-03 |
| **名称** | 往期概念说明 |
| **所属维度** | D4 结构完整性 |
| **判定标准** | 文中引用前文核心概念时，是否提供了简要说明 |
| **通过条件** | 引用前文概念时有 1-2 句话的简要说明，新读者也能理解 |
| **扣分** | 每处未说明扣 1 分（D4 维度） |

### S-04 承接上期

| 属性 | 内容 |
|------|------|
| **编号** | S-04 |
| **名称** | 承接上期 |
| **所属维度** | D2 开头吸引力 |
| **判定标准** | 非首篇文章开头是否自然承接上期 |
| **通过条件** | 开头有自然的前文引用（非模板化句式），且不依赖前文也能理解 |
| **扣分** | 缺失扣 1 分（D2 维度）；使用模板化句式（「在上期文章中」）扣 1 分 |

### S-05 身份一致性

| 属性 | 内容 |
|------|------|
| **编号** | S-05 |
| **名称** | 身份一致性 |
| **所属维度** | D9 事实准确性 |
| **判定标准** | 文中的用户身份描述是否与系列计划文件一致 |
| **通过条件** | 身份描述无矛盾（需对照系列计划文件检查） |
| **扣分** | 矛盾扣 2 分（D9 维度） |

### S-06 收官总指引

| 属性 | 内容 |
|------|------|
| **编号** | S-06 |
| **名称** | 收官总指引 |
| **所属维度** | D4 结构完整性 |
| **判定标准** | 系列最后一篇是否包含系列导航 |
| **通过条件** | 文末有「系列导航」段落，列出全部文章标题和一句话概括 |
| **扣分** | 缺失扣 3 分（D4 维度） |
| **适用** | 仅系列最后一篇检查 |

### S-07 系列标识

| 属性 | 内容 |
|------|------|
| **编号** | S-07 |
| **名称** | 系列标识 |
| **所属维度** | D4 结构完整性 |
| **判定标准** | 文章末尾是否有系列标识（系列名称 · 第X篇/共Y篇） |
| **通过条件** | 正文结束后、下期预告之前有分隔线和系列标识；开头和标题中无篇数标注 |
| **扣分** | 文末缺失扣 1 分（D4 维度）；开头或标题中出现篇数标注扣 1 分（D4 维度） |
| **适用** | 所有系列文章 |

### S-08 风险提示完整性（P23补丁）

| 属性 | 内容 |
|------|------|
| **编号** | S-08 |
| **名称** | 风险提示完整性 |
| **所属维度** | D9 事实准确性 |
| **判定标准** | 理财/投资类文章是否包含完整的风险提示 |
| **通过条件** | 文章末尾包含以下全部内容：<br>1. "不构成投资建议"或类似表述<br>2. "市场有风险，投资需谨慎"<br>3. "定投不保本"<br>4. "基金名称仅为举例，不构成推荐"（如提及具体基金名称） |
| **扣分** | 缺失任一项扣2分；全部缺失触发V-21否决 |
| **适用** | 理财访谈系列、投资理财类文章 |
| **检测方法** | 1. 判断文章是否属于理财/投资类（关键词：基金、股票、定投、理财、投资组合等）<br>2. 如果是，检查文章末尾是否存在标准风险提示<br>3. 对照V-21否决项检测完整性 |

---

## R2 执行流程

1. 确认文章是否属于系列文章（检查系列计划文件是否存在）
2. 如果是系列文章，按 S-01 到 S-08 逐项检查
3. 将扣分计入对应维度
4. 在 R2 报告中单独列出系列检查结果

---

## 与现有否决项的关系

本检查项**不构成一票否决**，而是作为维度扣分的补充。但如果 S-01（独立可读性）严重违反（多处假设读者看过前文），R2 可以在报告中建议 R4 降级处理。
