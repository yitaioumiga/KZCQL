## 三、W2 新补充信息的事实核查（新引入问题）

W2 为文章中提到的每个模型补充了具体的技术参数和发布信息。以下逐项验证。

### 3.1 Kimi K2.6 — 事实错误（严重）

**文章声称**（第65行）："1T参数的MoE架构，320B激活参数"

**实际事实**：Kimi K2.6 总参数为 1T，但激活参数为 **32B**（320亿），而非 320B。

- 证据1：DataLearner AI 记录"总参数量为1万亿（1T），每个token激活的参数量为320亿（32B）"
- 证据2：Kimi K2 官方博客"1T 总参数 / 32B 激活参数 / 384 专家"
- 证据3：CSDN 多篇技术分析文章均确认 1T/32B 架构

**错误性质**：320B 与 32B 差了一个数量级（10倍）。这是将"320亿"误写为"320B"（B=十亿，320B=3200亿），属于严重的事实错误。

**触发项**：V-10 事实核查失败

### 3.2 GLM-5.1 — 事实偏差（轻微）

**文章声称**（第63行）："编程能力距全球最强仅差3分"

**实际事实**：GLM-5.1 在 SWE-bench Pro 上得分 58.4%，**超越**了 GPT-5.4（57.7%）和 Claude Opus 4.6（57.3%），是 SWE-bench Pro 的全球第一。

- 证据1：CSDN 报道"GLM-5.1 以 58.4% 的 SWE-bench Pro 得分，登顶全球开源模型编程能力榜首，超越了 GPT-5.4 和 Claude Opus 4.6"
- 证据2：爱范儿微博提到编程评测得分 45.3 与 Claude Opus 4.6 的 47.9 差 2.6 分（非3分），但这是另一个评测维度

**错误性质**：GLM-5.1 在核心编程基准（SWE-bench Pro）上已经是全球最强，而非"差3分"。文章的表述与事实相反。

**触发项**：V-10 事实核查失败（轻微偏差）

### 3.3 文心5.0 — 准确

**文章声称**（第55行）："文心5.0，2.4万亿参数，1月22日发布的"、"原生全模态"

**实际事实**：
- 2.4万亿参数：IT之家、人民网、央广网等多方确认
- 1月22日发布：IT之家、人民网、央广网等多方确认
- 原生全模态统一建模：多方确认

**判定**：**准确**

### 3.4 豆包12.8.0 — 准确

**文章声称**（第61行）："豆包12.8.0，4月13日更新，这次搭载了Seeduplex全双工语音模型"

**实际事实**：
- 豆包12.8.0 版本号：应用商店确认
- 4月13日更新：今日头条等多方报道确认"2026年4月13日，字节跳动旗下智能助手豆包迎来12.8.0版本重大更新"
- Seeduplex 全双工语音：字节跳动官方 Seed 页面确认"2026年4月9日"发布

**判定**：**准确**

### 3.5 海螺AI M2.7 — 部分准确

**文章声称**（第57行）："MiniMax M2.7，3月到4月7日之间陆续更新"、"SWE-Pro上拿了56.22%"、"Terminal Bench 2打到了SOTA"

**实际事实**：
- 发布时间：M2.7 于 2026年3月18日正式发布，4月12日正式开源。文章称"3月到4月7日之间"——4月7日这个截止点没有明确对应事件，4月12日开源才是后续重要节点。**轻微偏差**。
- SWE-Pro 56.22%：多方确认，**准确**。
- Terminal Bench 2 SOTA：M2.7 发布时（3月18日）Terminal Bench 2 得分为 57.0%，当时可能是 SOTA。但到4月，Kimi K2.6 在 Terminal-Bench 2.0 上已达到 66.7%，M2.7 不再是 SOTA。文章写于4月24日，称"打到了SOTA"已不准确。**轻微偏差**。

**判定**：**部分准确**（发布时间区间和 SOTA 描述存在轻微偏差，但不构成严重事实错误）

### 3.6 Yi-1.5 — 准确

**文章声称**（第53行）："Yi-1.5是2024年发布的了，2026年4月到现在连个更新都没有"

**实际事实**：Yi-1.5 于 2024年5月13日正式开源（GitHub、HuggingFace、阿里云开发者社区均确认），截至2026年4月未发现 Yi 系列新版本发布。

**判定**：**准确**

### 3.7 通义千问 Qwen3.6 — 准确

**文章声称**（第69行）："Qwen3.6系列，4月2日到22日之间密集发布了Plus、Max-Preview和27B三个版本"

**实际事实**：
- Qwen3.6-Plus：2026年4月2日发布（CSDN、阿里云确认）
- Qwen3.6-Max-Preview：2026年4月22日发布（Alibaba Cloud Blog 确认）
- Qwen3.6-27B：2026年4月22日发布（IT之家、Qwen官方博客确认）

**判定**：**准确**

### PASS / FAIL：FAIL
