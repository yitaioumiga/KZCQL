4月23号那天，我的手机从早上开始就没停过。

先是OpenAI发了GPT-5.5，然后DeepSeek V4跟着就亮了。群里一排排截图往外甩，跑分表格、价格对比、技术架构图，信息密度大到让人脑仁疼。

但说实话，真正让我放下手里的活儿认真去刷的，不是这两个大佬同台竞技这件事本身。而是我翻了半天评测之后发现的一个很微妙的现象。

DeepSeek V4，好像不是最强的那个。

你敢信？？？

一个被整个中国AI圈寄予厚望的模型，发布之后跑分居然不是第一。这事儿搁在一个月前，谁信啊。

![配图1：四月的AI发布潮时间线](images/v4-cover.jpg)

我先把数据摊开来说，不藏着掖着。

Arena AI Code Leaderboard，这个排行榜在编程圈子里算是目前最权威的实时排名了，arena.ai上的数据，真实用户盲测打分，不是厂商自己报的。截至4月23号，排名是这样的。

GLM-5.1，Elo 1534，第一。Kimi K2.6，Elo 1529，第二。DeepSeek V4 Pro，Elo 1456，第三。

V4 Pro排在第三，跟第一的GLM-5.1差了78个Elo。在竞技排名里，78个Elo基本上意味着你能明显感知到实力差距。

然后我再去翻Vals AI的全量评测和DeepSeek官方技术报告，更细的基准测试数据出来了。

V4赢的项目确实有。LiveCodeBench，93.5%，全场第一，Kimi K2.6是89.6%，Gemini 3.1 Pro是91.7%。Codeforces评分3206，超过GPT-5.4的3168。Chinese-SimpleQA，84.4分，中文能力确实突出，Kimi K2.6是75.9，GPT-5.4是76.8。IMOAnswerBench，89.8，Kimi K2.6是86.0。

但V4输的项目也不少。SWE-bench Pro，Kimi K2.6拿了58.6，GLM-5.1是58.4，V4只有55.4。这个测试衡量的是真实工程能力，就是给你一个大型代码库，你能不能定位bug、写出正确的修复方案。V4在这个项目上输了。HLE w/tools，Kimi K2.6是54.0，V4是48.2，工具调用能力差了一截。Terminal-Bench 2.0，V4干脆没进前九。Vibe Code Bench，Claude Opus 4.7拿了71.0%，V4只有49.93%，从零构建Web应用的能力差了二十多个百分点。最离谱的是ProofBench，GPT-5.4是56%，V4只有10%，数学形式证明这个项目上差距悬殊到让人怀疑是不是同一个量级的模型。

![配图2：V4关键基准对比数据图表](images/v4-benchmark.jpg)

所以结论很清楚。V4不是最强的。在综合编程能力上，GLM和Kimi走在前面。在真实工程场景里，Kimi的SWE-bench Pro成绩更好。在形式化数学证明上，GPT-5.4遥遥领先。在Vibe Code Bench这种从零构建Web应用的测试上，Claude Opus 4.7比V4高了二十多个百分点。

我有时候觉得，DeepSeek这次发布，有点像那种考试考了第三名的学霸。你看着成绩单说，嗯，不错。但你心里知道，他上次可是考第一的。

不过话说回来，跑分这事儿吧，你得跳出成绩单去看这个人本身。

我刷完一圈评测之后，靠在椅背上，脑子里一直在想一件事。一个中国团队，用国产芯片、国产框架、国产数据，造出了一个能跟GPT-5.5同日发布的模型，在LiveCodeBench上拿了全场第一，在Codeforces上超过了GPT-5.4。

这件事本身有多难？

你可能不在AI行业里，对「国产芯片训练大模型」这件事没什么感觉。我换个说法你就懂了。

现在全球几乎所有的大模型，训练和推理用的都是NVIDIA的GPU，跑的都是CUDA这个软件框架。CUDA是NVIDIA花了将近二十年打造的生态，几乎所有的AI开发者都在用，所有的深度学习框架都针对CUDA做了极致优化。你要训练一个大模型，你几乎不可能绕开NVIDIA。

这玩意就像全世界所有电脑的操作系统都是Windows，你做软件只能适配Windows。你想用别的？对不起，整个软件生态都不支持你。

而且NVIDIA不只是卖硬件，它卖的是一整套生态锁。你用了它的GPU，就得用它的CUDA。你用了它的CUDA，你的代码就只能在它的硬件上跑。你想换？可以，把你所有的代码重写一遍，把所有的算子重新实现一遍，把所有的性能优化从头做一遍。这个迁移成本大到几乎没有团队能承受。

所以当DeepSeek说「去CUDA化」的时候，你如果了解这个行业，就知道这三个字的分量有多重。

![配图3：文艺插图 - 从依赖到自主](images/v4-localization.jpg)

很多朋友可能不知道，DeepSeek V4这次做了一个在整个AI行业里几乎没人做过的事。

全产业链国产化。

华为昇腾芯片，CANN框架，去CUDA化。从芯片到框架到模型，整条链路全部用国产方案。不是在某个环节上做替代，而是从底层开始，整套体系自己来。

愚钝如我，一开始看到这些信息的时候，也没太反应过来。但后来我顺着这个线索往下挖，越挖越觉得后背发凉。

你想想看，一年前还有人在说，国产芯片根本跑不了大模型训练，昇腾的性能差太远了，CANN的生态太烂了，去CUDA化就是自寻死路。现在DeepSeek V4就在那里，用这套「自寻死路」的方案训练出来的模型，跟GPT-5.5同一天发布，在好几个基准测试上跟全球最顶尖的模型打得有来有回。

事实胜于一切争论。

而且你要知道，用国产方案训练大模型，跟用NVIDIA的成熟方案训练大模型，难度完全不是一个量级。用NVIDIA的方案，你遇到问题，网上到处都是解决方案，Stack Overflow上随便一搜就有答案，GitHub上开源项目直接能跑。用国产方案呢？所有的坑都要自己踩，所有的bug都要自己修，所有的经验都要从头积累。别人踩过的坑你还得再踩一遍，因为环境不一样了。

我之前在技术社区看到一个帖子，发帖的人是一个在国产芯片上调试模型的工程师。他说他花了三个月时间，把一个在NVIDIA GPU上运行良好的模型迁移到了华为昇腾上。三个月里遇到的问题不计其数，有些算子在昇腾上没有对应的实现，有些性能瓶颈根本找不到原因，有些bug在CUDA上不会出现但在CANN上会复现。

他说最难的不是技术问题，是孤独。

因为几乎没有人做过这件事，网上找不到现成的解决方案，问同事也没有人遇到过类似的情况。他只能一个人，对着文档一行一行地啃，对着报错信息一条一条地排查。

三个月后，模型终于在昇腾上跑通了。他发帖说，跑通的那一刻，他盯着屏幕看了很久，然后关掉电脑，去楼下便利店买了一罐啤酒，一个人坐在路边喝完了。

我不知道他叫什么名字，但我能想象那个画面。深夜的便利店，一罐啤酒，一个人坐在路边，路灯把影子拉得很长。

他做的事情，可能不会出现在任何新闻里，不会有任何媒体报道，不会有任何跑分榜单记录他的贡献。但他踩过的每一个坑，积累的每一条经验，都变成了国产AI底座的一部分。

DeepSeek V4能在昇腾加CANN上跑起来，背后有无数个像他这样的人。

![配图4：文艺插图 - 自主可控](images/v4-sovereignty.jpg)

说到这块，我脑子里一直有一个画面挥之不去。

2020年8月，华为麒麟9000发布。

如果你还记得那个时间点的话，应该知道那是什么处境。华为已经被美国列入实体清单一年多了，芯片代工被切断，麒麟9000成了「绝唱」。所有人都知道，这颗芯片发布之后，可能很长一段时间内，华为都造不出新的旗舰芯片了。

而且说实话，麒麟9000的跑分也不是全球第一。

当时高通的骁龙888在GPU性能上更强，苹果的A14在单核性能上领先。麒麟9000在多项跑分上并不是最强的那个。但没有人会因为跑分不是第一就说麒麟9000不重要。

因为麒麟9000证明了一件事，中国芯片产业链可以独立运转。

从芯片设计到制造工艺，从指令集到操作系统，整条链路都能自己搞定。这个「能搞」本身，比跑分第一重要一万倍。跑分第一说明你跑得快，「能搞」说明你不会被别人卡脖子。

后来的事情大家都知道了。华为咬着牙搞出了鸿蒙，搞出了新的麒麟芯片供应链，Mate 60系列发布的时候，很多人排队去买，不是因为芯片性能吊打高通，是因为大家想用行动告诉华为，你撑住，我们在。

DeepSeek V4在大模型领域的意义，跟麒麟9000在芯片领域的意义是一样的。

V4的跑分不是第一，GLM和Kimi在多项测试上领先。但V4证明了一件事，中国可以独立造出前沿大模型。不用NVIDIA的GPU，不用CUDA，用国产芯片、国产框架、国产数据，照样能训练出世界级的AI模型。

这个「能造」本身，就是历史性的突破。

你想想看，大模型是未来十年最重要的基础设施之一。如果这条基础设施的底座完全依赖别人的芯片和框架，那等于把命脉交到了别人手里。哪天制裁一来，断供一断，整个行业就停摆了。

DeepSeek V4做的事情，就是在大模型领域把命脉握到了自己手里。就算有一天NVIDIA的GPU对中国完全断供，就算CUDA的更新对中国开发者完全关闭，DeepSeek依然能训练模型，依然能迭代，依然能服务用户。

跑分可以输，命脉不能断。

![配图5：文艺插图 - 麒麟9000与V4的历史对照](images/v4-models.jpg)

其实吧，这条路我们走过很多次了。

中国高铁。最早是引进的，从德国西门子、法国阿尔斯通、日本川崎重工那里买技术、学经验。当时很多人嘲笑，说中国只会模仿不会创新。但十几年过去了，中国高铁不仅学会了，还超越了。复兴号动车组，从牵引系统到网络控制到转向架，全部自主知识产权。现在中国高铁运营里程占全球三分之二以上，技术出口到印尼、沙特、土耳其。当年嘲笑你的人，现在排队来买你的技术。

中国航天。1990年代，国际空间站项目把中国排除在外。美国出台了沃尔夫条款，明确禁止NASA与中国航天合作。结果呢？中国自己建了天宫空间站，自己搞了北斗导航系统，自己的嫦娥探月、天问探火。被排除在外的那个人，最后自己盖了一栋更大的房子。而且天宫空间站现在向所有联合国会员国开放合作，格局直接拉满。

每一次，都不是最强的那个。但每一次，都把命脉握在了自己手里。

自主可控这条路，从来不是因为「我们最强」才走的，是因为「我们不能没有」才走的。

而且DeepSeek V4还有一个很重要的点，它是Apache 2.0开源的。整个国产AI生态都能站在这个肩膀上。其他团队可以在DeepSeek的基础上做优化、做定制、做垂直领域的应用，而不需要每个人都从零开始搭建国产化的底座。DeepSeek替大家把最难的那段路走通了。

这就像华为把鸿蒙的核心能力开放给其他厂商一样，一个人踩完坑，后面的人就不用再踩了。

反正我觉得，这才是让我真正兴奋的地方。不是跑分赢了，是底座换了。从今天开始，中国AI不再只是站在别人的地基上盖房子，我们有了一块自己的地。这块地可能还不够平整，上面的房子可能还不够漂亮，但它是我们自己的。

![配图6：数据图表 - V4价格与竞品对比](images/arch-v4-pricing.png)

说到这块需要注意一下，全产业链国产化还带来一个很自然的副产品，就是价格。

DeepSeek V4 Pro，输入每百万token 1.74美元，输出每百万token 3.48美元。GPT-5.5，输入5.00美元，输出30.00美元。Claude Opus 4.7，输入15.00美元，输出75.00美元。

V4 Pro的输出价格是GPT-5.5的八点六分之一，是Claude Opus 4.7的二十一分点五分之一。

你用Claude Opus 4.7处理一百万token的输出要花75美元，用V4 Pro只要3.48美元。

为什么能做到这个价格？因为全产业链在自己手里。你用NVIDIA的GPU，光硬件成本就是一笔巨款，再加上CUDA的授权费用、生态依赖带来的迁移成本，整个账算下来，成本根本压不下来。但如果你从芯片到框架到模型全是自己的，中间的每一层利润和每一层依赖成本都可以砍掉。

价格优势不是DeepSeek刻意打价格战，是国产化的自然结果。你把命脉握在自己手里了，成本自然就降下来了。

当然，V4也有明显的短板。不支持多模态输入，纯文本。在2026年4月这个时间节点，几乎所有一线模型都支持图片、视频、音频的多模态输入了，V4还只能处理文本。这个短板在产品体验上是很明显的，你没法直接丢一张截图给它说「帮我分析一下这个界面」，你得先把图转成文字描述再喂给它。

不是说文本模型不行，而是多模态已经成了2026年AI产品的标配能力，没有这个能力，在很多场景下就是少了一条腿。

但话说回来，多模态能力可以后面迭代加上去，底座换了才是根本性的变化。房子可以后面装修，但地基得先打在自己家的地上。

![配图7：文艺插图 - 底座与未来](images/arch-v4-overall.png)

回到开头那个场景。

4月23号，我的手机从早上开始就没停过。GPT-5.5和DeepSeek V4同日发布，群里一排排截图往外甩，跑分表格、价格对比、技术架构图。

我刷完一圈之后，靠在椅背上，朋友圈和社群里还在刷屏。各种跑分对比，各种「XX模型吊打XX模型」的标题。

但我的视角已经不一样了。

我看到的不再是好几个模型在比谁的分数高。我看到的是一条路，一条从依赖到自主的路。

你回看整个4月的时间线。4月1号，GLM-5.1发布。4月2号，Qwen 3.6 Plus。4月16号，Claude Opus 4.7。4月20号，Kimi K2.6。4月23号，DeepSeek V4和GPT-5.5同日发布。一个月之内，六大旗舰轮番轰炸，中国团队占了四个。

GLM-5.1在Arena排名上拿了第一，Kimi K2.6紧随其后，Qwen 3.6在综合实力上稳步推进。每个团队都在用自己的方式往前走。但DeepSeek V4做的事情不一样，它不是在跟别人比谁跑得更快，它是在修一条自己的跑道。

这条跑道修好之后，所有人都能在上面跑。

我有时候会想，五年后再回头看2026年4月，人们会记住什么？会记住GLM-5.1的Elo 1534吗？会记住Kimi K2.6的SWE-bench Pro 58.6吗？

可能不会。

但人们会记住，2026年4月，有一个中国团队，用国产芯片、国产框架、国产数据，造出了一个能跟GPT-5.5同台竞技的大模型，并且把它开源给了全世界。

就像人们记住麒麟9000，不是因为它的跑分，而是因为它证明了中国芯片产业链可以独立运转。

今天不是中国AI最强的一天。

但可能是最重要的一天。

以上，既然看到这里了，如果觉得不错，随手点个赞、在看、转发三连吧，如果想第一时间收到推送，也可以给我个星标。

谢谢你看我的文章，我们，下次再见。
