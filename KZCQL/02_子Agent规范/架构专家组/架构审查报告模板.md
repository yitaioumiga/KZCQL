## 评分等级标准

| 等级 | 分数范围 | 说明 |
|------|----------|------|
| A+ | 108-120 | 系统非常健康 |
| A | 96-107 | 系统健康 |
| B+ | 84-95 | 系统基本健康，有小问题 |
| B- | 72-83 | 系统存在明显问题 |
| C | 60-71 | 系统存在严重问题 |
| D | <60 | 系统不健康 |
