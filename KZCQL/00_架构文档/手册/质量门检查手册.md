## 五、主Agent 检查清单

### 5.1 阶段转换检查

每次阶段转换前，主Agent 必须确认：

**阶段零 → 阶段一**：
- [ ] 所有规则文件已加载
- [ ] 工作区已就绪
- [ ] khazix-writer Skill 可用

**阶段一 → 阶段二**：
- [ ] 写作前必问 5 项已全部确认
- [ ] W1 已通过 khazix-writer Skill 输出初稿
- [ ] 初稿已保存至正确路径

**阶段二 → 阶段三**：
- [ ] R1 评审由独立子代理执行（非主代理自行评审）
- [ ] R2 评审由独立子代理执行（或因 R1=FAIL 被正确跳过）
- [ ] R4 综合评级由独立子代理执行并输出

**阶段三 → 阶段四**：
- [ ] 评级路由决策已做出
- [ ] W2 或 W1 已收到正确的输入包

**阶段四 → 阶段五**：
- [ ] 退出条件已触发（P0/P1/P2）
- [ ] 最终版本的 R4 评级报告已生成且保存（无R4报告不得进入阶段五）
- [ ] 最终版本已保存

**阶段五 → 完成**：
- [ ] 用户已确认满意（或已收集反馈）
- [ ] A1 架构审查已完成
- [ ] E1 规则修改已完成（如有必要）
- [ ] 所有文件已保存至正确路径
- [ ] 归档完成流程已执行（通用归档规则 §五 的 5 个步骤全部完成）
- [ ] 归档目录结构已验证（稿件/配图/评审三个子目录均存在且内容完整）

### 5.2 质量门检查

主Agent 在向用户交付前，必须通过以下质量门：

| 检查项 | 标准 | 不通过时的处理 |
|--------|------|----------------|
| 文章版本号正确 | 符合命名规范 | 修正版本号 |
| 评审报告完整 | 包含所有必要字段 | 要求评审 Agent 补充 |
| 评级来自 R4 | 非主Agent 自行判断 | 重新调用 R4 |
| 退出条件已触发 | 符合 P0/P1/P2 标准 | 继续迭代 |
| 文件路径正确 | 符合路径规范 | 移动文件 |
| 用户反馈已收集 | 用户已确认或提供反馈 | 等待用户反馈 |
| 配图已生成且通过验证 | 使用 Read 工具逐张检查，确认无水印、无文字、风格统一 | 重新生成不合格的图片 |
| 架构图使用 HTML 渲染 | 架构图文件为 .png 格式且中文清晰可读 | 使用 Playwright/wkhtmltoimage 重新渲染 |
| **数据时效性检查**（P20补丁） | 时间敏感数据标注时间与当前时间差距不超过阈值，或标注"历史数据" | 使用WebSearch获取最新数据并更新 |
| 发布前检查清单全部通过 | 逐项检查 18 项检查项 | 返回对应 Agent 修复 |
